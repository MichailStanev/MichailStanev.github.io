Simple Tasker

Status: to be released

Github

This is an android based project that aims to simplify the automation of day to day tasks, using the google awareness API and MongoDB stitch. The main objective of the app is to allow the user to create both simple and complex tasks.The phone does an action (Put phone is silent mode or send user a reminder), based on gps coordinates, specific time or during some activity(running, walking, etc)

What is implemented at the moment:

  • Connection to the MongoDB database
  • The BroadcastReciever that listens for activation or deactivation of specific fence
  • Creation of a geoFence
  • Action to mute/unmute phone on geoFence trigger

What is planned to be implemented in the future:

  • Action activate/disable wi-fy/mobileData
  • Action giving user reminders based on fence responce
  • Creation of time fences
  • Creation of action fences (running, walking, stationary, in a car, driving)
  • Combination of multiple fences
